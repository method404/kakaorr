# Kakaorr v0.1.0

## 요약

첫 공개 버전입니다. 카카오페이지 웹툰 개인 수집과 라이브러리 관리를 위한 기본 관리자 UI와 저장 흐름을 정리했습니다.

## 포함 내용

- 라이브러리, 추가하기, 요일별, 설정 화면 구성
- 카카오페이지 작품 검색과 요일별 탐색 기능 추가
- 작품 메타데이터 및 회차별 이미지 저장 구조 구현
- 회차별 저장 상태, 수동 재시도, 상세 정보 화면 구현
- 카카오 세션 업로드 기반 로그인 브리지 추가
- 기다무 회차 자동 해금 시도 및 다음 해금 시각 예약 로직 반영
- Docker 실행 구성과 데이터/스토리지 분리 볼륨 구성 추가

## 주의사항

- 아직 개발 중인 초기 버전입니다.
- 카카오 로그인 및 성인 콘텐츠 접근은 외부 정책 변화에 영향을 받을 수 있습니다.
- 자동화 스케줄, 인증 복구, 배포 업데이트는 계속 보완 중입니다.

## 면책

- 이 프로젝트는 개인 저장용입니다.
- 수집 결과물의 공유, 재배포, 상업적 사용은 사용자 책임입니다.
- 계정 제한, 데이터 손실, 서비스 차단, 법적 문제 등 사용 중 발생하는 모든 문제는 사용자 본인 책임입니다.
